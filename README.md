## 2. 定义合理边界——版本

边界，代表一类训练的范围，同时也决定着训练能带来的效果。这里使用版本梯度来表示边界。

为每一类训练指定一个合理的版本以及版本内容的规划。

训练评测——能够分享，代表着对一个训练的掌握。常见的自话式分享，PPT式分享。

定义一个合理的版本目标。
